A Procreate-friendly palette sheet generator for illustrator workflows.

## What this is

qisi-color-kit helps artists build labelled color sheets for character illustration.

Workflow:

lineart -> color palette -> Procreate color picking -> finished illustration

## Features

- Labelled color swatches
- Skin / hair / eye / clothing / lineart categories
- HEX color input
- Large color blocks for direct Procreate color picking
- PNG palette export
- Markdown export
- Built for artist portfolio / GitHub presentation use
